DP WORLD MARITIME FP&A CHATBOT
================================

APPLICATION IS READY

TO START:
---------
python app.py

Server runs on: http://localhost:5001


QUERY EXAMPLES:
---------------

1. Direct Retrieval - Variance/Comparison
   "Charter Hire actual vs budget variance for Africa"
   Returns: Actual amount, Budget amount, Variance calculation

2. Total by Vessel
   "total costs by vessel"
   "show me costs per vessel"
   Returns: Total costs for each vessel in the unit

3. Financial Ratios
   "crew to revenue ratio"
   Returns: Crew cost divided by total revenue for each vessel

4. Category Totals
   "how much is crew cost"
   "total charter hire expenses"
   Returns: Total actual amount for the specified category


API ENDPOINT:
-------------
POST /api/query

Request:
{
  "message": "your financial question",
  "unit": "Africa"  (default is Africa, can also be "MENA")
}

Response:
{
  "success": true,
  "query": "original question",
  "answer": "AI-generated narrative explanation",
  "table": [...array of result rows...],
  "metadata": {
    "rows": number of results,
    "unit": "Africa",
    "timestamp": ISO timestamp
  }
}


SUPPORTED QUERY TYPES:
---------------------
✓ Direct retrieval (totals, by vessel)
✓ Actual vs Budget variance
✓ Ratios (crew/revenue, etc.)
✓ Comparisons across vessels
✓ Financial category analysis

All results include:
- Narrative answer generated by LLM
- Complete data table
- Metadata (rows returned, unit, timestamp)


DATABASE:
---------
financial_data.db
Tables: monthly_financials, vessels, units, pl_categories
112 financial categories tracked
Africa and MENA units
Multiple vessels per unit
Actual, Budget, Last Year, and Risk Factor values
